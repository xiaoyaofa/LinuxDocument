## media-ctl
打印设备拓扑
```
media-ctl -d /dev/media0 --print-topology
```

## v4l2-ctl
列出设备支持的图像格式
```
v4l2-ctl -d /dev/video0 --list-formats
```
列出支持的格式和分辨率
```
v4l2-ctl -d /dev/video0 --list-formats-ext
```
获取配置
```
v4l2-ctl -d /dev/video0 --get-fmt-video
```
捕获原始图像
```
v4l2-ctl --stream-mmap --stream-count=1 --stream-to=frame.raw
```
