## Debian/Ubuntu
apt install xinput
export DISPLAY=:0
xinput list
找到触摸屏对应的ID号
xinput --list-props <ID> DISPLAY=:0
这里发现使用libinput

vi /usr/share/X11/xorg.conf.d/40-libinput.conf
找到touchscreen，添加option
```
Section "InputClass"
        Identifier "libinput touchscreen catchall"
        MatchIsTouchscreen "on"
        MatchDevicePath "/dev/input/event*"
        Driver "libinput"
        #Option "TransformationMatrix" "-1 0 1 0 -1 1 0 0 1"    #显示
        Option "CalibrationMatrix" "-1 0 0 0 -1 0 0 0 -1"       #触摸
EndSection
```
Matrix 参数说明
```
| a  b  c |
| d  e  f |
| 0  0  1 |
a: X 缩放
b: XY 倾斜
c: X 偏移
d: YX 倾斜
e: Y 缩放
f: Y 偏移
```
举例
反转 Y 轴：如上所示 0 -1 1
反转 X 轴：-1 0 1
同时反转 X 和 Y：-1 0 1 0 -1 1 0 0 1
