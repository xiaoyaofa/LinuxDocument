## 设备树配置
imx8mp举例，只需要一个gpio
```
ir_recv: ir-receiver {
    compatible = "gpio-ir-receiver";
    gpios = <&gpio4 8 GPIO_ACTIVE_LOW>;
    pinctrl-names = "default";
    pinctrl-0 = <&pinctrl_ir_recv>;
    linux,autosuspend-period = <125>;
    status = "okay";
};
pinctrl_ir_recv: ir-recv {
    fsl,pins = <
            MX8MP_IOMUXC_SAI1_RXD6__GPIO4_IO08              0x4f
    >;
};
```
## 测试
使用v4l-utils中的ir-keytable
如果不确定红外协议，就先启动所以协议
```
ir-keytable -p all
```
监控红外协议
```
ir-keytable -t
```
