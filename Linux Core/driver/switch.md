## 拓扑图
下面为简单的交换机芯片连接拓扑图
功能1：就是基本的转发功能测试，链路为 Dev Board 1 <--> Switch Chip <--> Dev Board 2
功能2：lan口测试，链路为 CPU <--> Switch Chip <--> Dev Board 1/2

```
    +-----------------------+
    |          CPU          |
    +-----------------------+
                ▲
                │
                │ RGMII/GMII (MAC-PHY)
                ▼
+--------------------------------+
|          Switch Chip           |
|                                |
+--------------------------------+
       ▲                  ▲
       │                  │
       │ Ethernet         │ Ethernet
       ▼                  ▼
+-------------+    +-------------+
| Dev Board 1 |    | Dev Board 2 |
+-------------+    +-------------+
```
